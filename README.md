El proyecto utiliza dependencias PHP gestionadas con Composer.
Estas dependencias se instalan directamente en el servidor y no se incluyen en el repositorio por buenas prácticas de seguridad y mantenimiento.

Instalación de dependencias (Composer)
El proyecto usa Composer para manejar dependencias PHP (dompdf).
